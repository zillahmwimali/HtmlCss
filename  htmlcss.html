CSS Box Model.

What is a box model?

CSS Box model is a container that contains a number of properties which include;
Padding-This is the space between an element's content and its border
Margin-This property controls the space outside the element
Content-This is the information of the web page.e.g the images and whatnot
Width-Width specifies the width of the elements and its content area
Border-This is basically the visible area enclosing the element
All these are essential in creating the layout and design of a web page.

Width:400px

Padding:20px*2=40px

Margin:20px*2=40px

Border:2px*2=4px

Total width=(400+40+40+4);484px
